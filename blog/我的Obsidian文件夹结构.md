# 我的 Obsidian 文件夹结构

## 顶层文件夹
📁 _inbox/ # 快速捕获，未处理
📁 projects/ # 项目笔记
📁 areas/ # 长期关注的领域
📁 resources/ # 参考资料
📁 archive/ # 归档


## 为什么这样设计

- `_inbox` 保证零门槛记录
- `projects` 有始有终
- `areas` 持续更新
- `resources` 只读不写
- `archive` 让主文件夹保持清爽

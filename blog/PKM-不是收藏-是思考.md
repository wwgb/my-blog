# PKM 不是收藏，是思考

很多人做笔记只是收藏文章，这不叫知识管理。

真正的 PKM 是：输入 → 理解 → 整理 → 输出 → 迭代。

笔记是为了让自己更聪明，不是为了装满文件夹。
